# Sistema de Conciliação Financeira

Sistema web desenvolvido para realizar:

- Conciliação de vendas de cartão
- Controle de recebimentos das operadoras
- Comparação automática entre vendas e pagamentos
- Gestão de empresas
- Gestão de operadoras/adquirentes

## Tecnologias utilizadas

- Node.js
- Express
- PostgreSQL
- HTML
- CSS
- JavaScript

## Funcionalidades atuais

✅ Cadastro de empresas  
✅ Cadastro de operadoras  
✅ Registro de vendas de cartão  
✅ Registro de recebimentos  
✅ Conciliação automática  
✅ Frontend integrado ao backend  
✅ Dashboard inicial  

## Objetivo do projeto

O objetivo do sistema é auxiliar supermercados e empresas na conferência financeira das vendas realizadas em maquininhas de cartão, identificando divergências entre valores vendidos e valores recebidos pelas operadoras.
